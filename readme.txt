Display latest movies and tv shows for selected tv channels available in Serbia.

== Description ==

Mojtv.net Wordpress plug in can display selected Mojtv.net feed on a page or post where [mojtv] short code is used or in sidebar where MojTv widget is used.
This plug in is in Serbian language.

== Installation ==

1.	Upload mojtv directory to /wp-content/plugins/ directory
2.	Activate plugin
3.	Settings are under Settings -> MojTv Settings

== Frequently Asked Questions ==

= How to use? =

On settings page user can choose what to include in the feed and save settings. After settings were choosen user can put [mojtv] short code on specific page or post or put widget into specific sidebar.
